Getting started

Include JS and CSS files

  1. Download HV-Autocomplete plugin js and css files.
  2. Add these files in header section of your page.
<!-- JS file -->
<script src="path/to/jquery.hv-autocomplete.js"></script>

<!-- CSS file -->
<link rel="stylesheet" href="path/to/hv-autocomplete.css">

HV-Autocomplete is dependent from jQuery. Please add jQuery file or use CDN server.

<!-- Using jQuery with a CDN -->
<script src="//code.jquery.com/jquery-1.11.2.min.js"></script>

Basic

HV-Autocomplete plugin is very simple to use. Just call function HVAutocomplete on a
<div id="basic"></div> field and give it a list of elements to display.

Example

HTML

  <div id="basics"></div>

JS

  var options = {
    data: countries,
    placeholder: "Search country...",
    maxLength: 5
  };

  $('#basics').HVAutocomplete(options);

Search

HV-Autocomplete has two search methods for your convenience.

Default search

If you want to use default search just use default options. Is very simple.

JS

  var options = {
    data: countries,
    placeholder: "Search country..."
  };

  $('#default-search').HVAutocomplete(options);

Global search

Global search looking all matches after space. For this search use globalSearch: true

JS

  var options = {
    data: countries,
    placeholder: "Search country...",
    maxLength: 5,
    globalSearch: true
  };
  
  $('#basics').HVAutocomplete(options);

Styling

Class

We tried to make the most convenient to stylize your autocomplete. So, you can add yourself class for input and result. Just add two options is inputClassName and/or resultClassName.

JS

  var options = {
    data: data,
    placeholder: "Search...",
    categories: true,
    inputClassName: "example-class-input",
    resultClassName: "example-class-result"
  };

  $("#class").HVAutocomplete(options);

Style

Same you can add style in tag with help inputStyles and/or resultStyles options

JS

  var options = {
    data: data,
    placeholder: "Search...",
    inputStyles: {
    paddingLeft: "30px",
    width: 300
    },
    resultStyles: {
    backgroundColor: "#efefef",
    width: 300}
  };

  $("#styles").HVAutocomplete(options);

Categories

With HV-Autocomplete you can separate your data to categories. For this use
categories: true option.

Vertical

If you want to get vertical(common) autocomplete, just use default options.

JS

  var options = {
    data: data,
    placeholder: "Search...",
    categories: true
  };
  
  $("#vertical").HVAutocomplete(options);

Horizontal

We want that user can use HV-Autocomplete in horizontal orientation. For this simple add displayHorizontal: true options. Same you can use your classes. See example above.

JS

  var options = {
    data: data,
    placeholder: "Search...",
    categories: true,
    displayHorizontal: true,
    globalSearch: true
  };

  $("#horizontal").HVAutocomplete(options);

API

Option Default value Type Description
data null object Format data: 
[
  {name: "name", url: "url"},
  {name: "name2", url: "url2"}
]
placeholder " " string Set placeholder for input.
maxLength not limited number Set maximum search result.
globalSearch false boolean true : search all matches after space;
false : default search.
inputClassName
 " " string Set class for input.
empty string : set default class name;
"your-class" : change default class to your.
resultClassName
 " " string Set class for result.
empty string : set default class name;
"your-class" : change default class to your.
inputStyles
 null object Set inline styles for input.
null : set default inline styles;
object : cahnge default inline styles to your.
resultStyles
 null object Set inline styles for result.
null : set default inline styles;
object : cahnge default inline styles to your.
categories
 false boolean true : set categories for autocomplete;
false : set default list.
Format data for categories:
{
  bmw: [
    {name: "I8", url: "url"},
    {name: "X5", url: "url"}
  ],
  mersedes: [
    {name: "SLK", url: "url"},
    {name: "Citan", url: "url"}
  ]
}
Name category get from name array.
displayHorizontal
 false boolean You can use this option only if you use categories option.
true : set your autocomplete horizontal
(adding inline style display: flex);
object : cahnge default inline styles to your.
